PE-ON V13.2 PERSONAL - PC 한 줄 조회 / 모바일·태블릿 확대·이동

적용 내용
1. PC 조회바: 학년-반 / 평가영역(또는 영역) / 조회를 한 줄로 표시
2. Z Flip6·Galaxy Tab S8: 1180px 작업 화면 전체를 처음에 축소 표시
3. 두 손가락으로 최대 5배 확대·축소 가능
4. 확대 상태에서 상하좌우 이동 가능
5. 모바일·태블릿의 상단바/조회바/탭 고정 해제
6. 사이드바는 기본 닫힘, 햄버거 버튼을 눌렀을 때만 오버레이로 표시
7. 버전 표기 V13.2 PERSONAL

배포
PowerShell에서 프로젝트 폴더를 연 뒤:
  npm install
  npm run build
  firebase deploy --only hosting
