PE-ON 수정 내용

1. 교과세특 오른쪽 ? 사용안내 문구/버튼 제거
   - JSX 파일에서 직접 제거
   - CSS에서도 혹시 남은 안내 버튼을 숨김 처리

2. 모바일 학급 전체 생성 생성/복사/삭제 버튼 크기 복원
   - 높이 54px
   - 폰트 16px
   - 버튼 3개 균등 배치

3. 모바일 500byte 옆 작은 버튼은 기존 작은 크기 유지

1. 압축을 풉니다.
