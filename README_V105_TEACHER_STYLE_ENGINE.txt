PE-ON V10.5 교과세특 교사 문체 엔진

- 학생 특성 자동 분류: 운동기능 우수형, 리더십형, 배려형, 책임감형, 노력형, 성장형, 균형형
- 문장 흐름: 학생 특성 → 수업 태도 → 수행 근거 → 성장 과정 → 학생별 관찰 → 종합 평가
- 예시 문체:
  기본 운동능력이 뛰어난 학생으로 운동 기술을 빠르게 습득하며 안정적인 수행을 보임.
  자신의 능력을 과신하지 않고 늘 배우려는 자세를 지니고 있으며 기능 향상을 위해 꾸준히 노력함.
- 수행평가 점수는 직접 노출하지 않고 문장 방향에만 반영
- 기존 UI와 기능 유지
- JavaScript/JSX 중괄호 균형 검사 완료

적용:
npm run build
firebase deploy --only hosting
