PE-ON V12.0 LTS

주요 반영 사항
- 체크형(평가요소) 수행평가 추가
- 평가요소 설정 팝업: 저장/수정/삭제, 우측 상단 X 고정
- 학생별 성공 요소 체크 후 자동 점수 계산 및 저장
- 평가요소별 성공 여부를 데이터로 보존해 세특 연계 가능
- 기존 시간/거리/횟수/선택형/직접점수 및 최종환산표 유지
- PAPS 결과 계산을 학생카드 기준과 통일한 수정 포함
- PC/태블릿/모바일 반응형 스타일 포함

실행
1. 이 폴더를 VS Code로 엽니다.
2. 터미널에서 npm install
3. npm run build
4. firebase deploy --only hosting
