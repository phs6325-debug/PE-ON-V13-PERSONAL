PE-ON 교과세특 v4 태블릿 버튼 보정

원인
- 삼성 태블릿 Chrome은 가로 화면에서 viewport 폭을 PC처럼 크게 잡는 경우가 있습니다.
- 그래서 max-width 기반 태블릿 CSS가 적용되지 않고, 버튼이 AI 결과 카드 아래쪽으로 밀려 보이지 않았습니다.

수정
1. PC: 기존처럼 AI 생성 결과 아래 버튼 유지
2. 모바일: 기존처럼 AI 생성 결과 아래 2×2 유지
3. 터치 태블릿: 5번 AI 생성 결과 제목 오른쪽에 생성/저장/취소/삭제 4개 버튼을 한 줄로 표시
4. 태블릿에서는 아래 버튼을 숨겨 중복 표시를 막음

적용
npm run build
firebase deploy --only hosting
