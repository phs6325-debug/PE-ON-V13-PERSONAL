PE-ON V13.30 모바일 월간 수업관리 및 웹 업데이트

적용 방법
1. 압축을 풉니다.
2. 안의 파일과 폴더를 기존 PE-ON 프로젝트 최상위 폴더에 그대로 덮어씁니다.
3. 아래 명령을 실행합니다.

npm install
npm run build
firebase deploy --only hosting

적용 기능
- 새 배포 버전 자동 감지
- 새 버전 발견 시 '지금 업데이트 / 나중에' 안내창 표시
- 상단의 '업데이트 확인' 버튼으로 수동 확인
- 앱을 다시 열거나 화면으로 돌아올 때 자동 확인
- 30분마다 자동 확인
- 업데이트 버튼을 누르면 새 캐시 적용 후 자동 새로고침
- sw.js/index.html 캐시 방지 설정
- 버전 표시 V13.30 PERSONAL
- 휴대폰 월간 수업관리 모달 수정 포함

주의
- 최초 1회 V13.28 배포 후에는 기존 설치 앱에서 브라우저 새로고침을 한 번 해야
  새 업데이트 기능 자체가 설치될 수 있습니다.
- 이후 V13.30부터는 앱 안에서 업데이트 안내가 자동 표시됩니다.
