PE-ON 교과세특 완전 재작성 v3

핵심 변경
1. StudentRecords.jsx가 seteuk-rebuild-v3.css를 직접 import합니다.
   → style.css 내부 충돌/깨짐 영향을 받지 않도록 분리했습니다.
2. 학생별 작성 화면을 sr3 전용 클래스 구조로 새로 만들었습니다.
3. 학생 목록이 다시 카드/버튼 형태로 표시됩니다.
4. 1~5번 카드 구조:
   1 수행평가 근거
   2 성장 과정
   3 학생별 메모
   4 공통 수업 내용
   5 AI 생성 결과
5. 버튼은 AI 생성 결과 아래에 위치합니다.
   - PC/태블릿: 4개 한 줄
   - 모바일: 2×2
6. 기존 학급 전체 생성 기능은 유지하고, 수행근거 %만 삭제했습니다.

적용 방법
1. 압축을 풉니다.
2. 기존 V9-개인전용 폴더에 덮어씁니다.
3. npm run build
4. firebase deploy --only hosting
