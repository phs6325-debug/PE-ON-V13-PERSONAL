PE-ON 학급 전체 생성 모바일 안전 복구

수정 내용
1. 학급 전체 생성 화면 구조는 기존 JSX 그대로 유지했습니다.
2. 모바일에서 PC형 표가 보이지 않도록 카드형 행 레이아웃을 강제로 복구했습니다.
3. 생성/복사/삭제 메인 버튼은 58px, 16px로 큰 버튼 복구했습니다.
4. 500byte 옆 작은 버튼은 작은 크기로 유지했습니다.
5. 태블릿의 수행근거/성장과정/생성세특 폭 조정도 유지했습니다.
6. ? 사용안내 숨김 처리했습니다.

적용 방법
npm run build
firebase deploy --only hosting
