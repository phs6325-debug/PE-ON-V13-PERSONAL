PE-ON V13.21 기본 설정 온라인/오프라인 동기화

적용 항목
- 학년도, 학기, 학년
- 저장한 기본 설정 / 현재 설정 유지
- 기본 시간표

동작 방식
1. 온라인
   - 기본 설정을 변경하면 Firebase Firestore에 자동 동기화합니다.
   - 다른 기기에서 로그인하면 서버의 기본 설정을 불러옵니다.

2. 오프라인
   - 변경 내용은 현재 기기의 localStorage에만 저장합니다.
   - 상단 상태가 '오프라인 · 기기 저장'으로 표시됩니다.

3. 인터넷 재연결
   - 오프라인 저장 자료가 있으면 동기화 확인 창을 표시합니다.
   - '지금 동기화': 기기 자료를 Firebase에 업로드합니다.
   - '나중에': 현재 기기에 유지하고 다음 접속 때 다시 확인합니다.

Firebase 문서 위치
peonUsers/{uid}/settings/basicSync
